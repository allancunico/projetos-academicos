

# Índices para Arquivos de Dados (C)

Este projeto implementa estruturas de índices em arquivos para permitir o acesso eficiente a grandes volumes de dados que não cabem totalmente na memória.

A abordagem utiliza arquivos auxiliares de índice para localizar registros específicos sem a necessidade de carregar todo o arquivo principal em memória, simulando técnicas utilizadas em sistemas de banco de dados.
